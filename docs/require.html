System Requirements

Linux

rpCalc requires the following libraries:

  • Python (Version 3.4 or higher)
  • Qt (Version 5.4 or higher - see Qt Project for more information)
  • PyQt (Version 5.4 or higher - see Riverbank for more information)

Windows

Using the files provided in the binary distribution, rpCalc should run on any computer running Windows XP, Vista, 7 or 8.
